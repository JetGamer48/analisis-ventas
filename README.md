# Proyecto TP Organización Empresarial
## Descripción
Este proyecto corresponde al trabajo práctico de Organización Empresarial. Incluye análisis de ventas y gestión de datos.
## Estructura del repositorio
- data: datasets utilizados
- scripts: código de análisis
- docs: documentación y resultados
## Integración con Jira
Cada commit está vinculado a una tarea Jira (ejemplo: PROY-1, PROY-2).
